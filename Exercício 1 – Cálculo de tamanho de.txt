Exercício 1 – Cálculo de tamanho de fonte em pixels

Temos o CSS:

html { font-size: 20px; }

html h1 { font-size: 2rem; }

h1 > .principal { font-size: 1.4em; }

html .secundario { font-size: 1.4rem; }


-----------//-----//-----//----//-----

*Lembrando:

*px = pixels (valor absoluto)

*rem = relativo ao font-size do root (html)

*em = relativo ao font-size do elemento pai

-------//-----//----------//------//------

1️⃣ h1

html h1 { font-size: 2rem; }

1rem = font-size do html = 20px

2rem = 2 × 20px = 40px

✅ h1 = 40px
--------//-------//------//-----//-------

2️⃣ .principal (filho de h1)

h1 > .principal { font-size: 1.4em; }
1em = font-size do pai

Pai (h1) = 40px

1.4em = 1.4 × 40px = 56px

✅ .principal = 56px
//----------//---------------//


3️⃣ .secundário

html .secundario { font-size: 1.4rem; }

1rem = font-size do html = 20px

1.4rem = 1.4 × 20px = 28px

✅ .secundario = 28px

-----//-------//-------//-------//-----//

| Elemento    | Tamanho (px) |
| ----------- | ------------ |
| h1          | 40px         |
| .principal  | 56px         |
| .secundario | 28px         |
